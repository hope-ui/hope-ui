# Installation

hope-ui is a batteries-included, themed, accessible component library for SolidJS.
Components ship ready to use and are styled through a Tailwind v4 preset, so you get
a coherent look out of the box while keeping full control over your design tokens.

## Install

```bash
pnpm add @hope-ui/components @hope-ui/theming @hope-ui/presets
```

## Set up the provider

Wrap your app in a `ThemeProvider` with a preset, then import components from their
own subpath:

```tsx title="App.tsx" showLineNumbers {8-10}
import { ThemeProvider } from "@hope-ui/theming";
import { hope } from "@hope-ui/presets/hope";
import { Button } from "@hope-ui/components/button";

function App() {
  return (
    <ThemeProvider preset={hope}>
      <Button variant="solid" colorScheme="primary">
        Click me
      </Button>
    </ThemeProvider>
  );
}
```
