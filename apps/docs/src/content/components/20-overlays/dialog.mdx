# Dialog

A modal dialog that traps focus, blocks the page behind a backdrop, and is fully
keyboard- and screen-reader-accessible. Built on the headless
`@hope-ui/primitives` dialog kernel.

## Usage

```tsx title="Dialog"
import { Dialog } from "@hope-ui/components/dialog";

<Dialog>
  <Dialog.Trigger>Open</Dialog.Trigger>
  <Dialog.Portal>
    <Dialog.Backdrop />
    <Dialog.Popup>
      <Dialog.Title>Confirm</Dialog.Title>
      <Dialog.Description>This action can't be undone.</Dialog.Description>
      <Dialog.Close>Cancel</Dialog.Close>
    </Dialog.Popup>
  </Dialog.Portal>
</Dialog>;
```

## Accessibility

- Focus is trapped inside the popup while open and restored to the trigger on close.
- Content outside the popup is marked `inert` and `aria-hidden`.
- `Escape` and a backdrop click dismiss the dialog.
