# `MESSAGES_ES` — Spanish catalog

Built-in **Spanish** catalog, selected when the active locale's primary subtag is `es` (`es`, `es-ES`,
`es-MX`, …). Shape + key list: `../messages.md`. Selection + fallback: `catalogs.md`.

- `datesSelected` plural: singular only at `1`, with feminine agreement (`1 fecha seleccionada` /
  `2 fechas seleccionadas`).
