# `MESSAGES_EL` — Greek catalog

Built-in **Greek** catalog, selected when the active locale's primary subtag is `el` (`el`, `el-GR`).
Shape + key list: `../messages.md`. Selection + fallback: `catalogs.md`.

- `datesSelected` plural: singular only at `1`, with feminine agreement on the adjective and noun
  (`1 επιλεγμένη ημερομηνία` / `2 επιλεγμένες ημερομηνίες`).
