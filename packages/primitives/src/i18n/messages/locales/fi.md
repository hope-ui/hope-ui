# `MESSAGES_FI` — Finnish catalog

Built-in **Finnish** catalog, selected when the active locale's primary subtag is `fi`. Shape + key
list: `../messages.md`. Selection + fallback: `catalogs.md`.

- `datesSelected` plural: `1` takes the nominative (`päivämäärä`); any other count takes the partitive
  singular (`päivämäärää`) — `1 päivämäärä valittu` / `2 päivämäärää valittu`.
- Ranges use an en dash (`{{start}}–{{end}}`), the Finnish range separator.
