# `MESSAGES_SV` — Swedish catalog

Built-in **Swedish** catalog, selected when the active locale's primary subtag is `sv`. Shape + key
list: `../messages.md`. Selection + fallback: `catalogs.md`.

- The neuter noun *datum* is invariant across number, so `datesSelected` inflects only the participle:
  `1 datum valt` (neuter singular) / `2 datum valda` (plural).
